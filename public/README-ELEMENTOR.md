## 📥 Cara Import Template ke Elementor

### Metode 1: Import Melalui WordPress Dashboard

1. **Login ke WordPress Admin**
   - Masuk ke dashboard WordPress Anda

2. **Buka Elementor Templates**
   - Navigasi ke: `Templates` → `Saved Templates`

3. **Import Template**
   - Klik tombol **"Import Templates"** di bagian atas
   - Pilih file: `elementor-hut-ri-81-template.json`
   - Klik **"Import Now"**

4. **Insert Template ke Halaman**
   - Buat halaman baru atau edit halaman yang sudah ada
   - Klik **"Edit with Elementor"**
   - Klik icon **Folder** (Add Template)
   - Pilih tab **"My Templates"**
   - Cari template "HUT RI Ke-81 - Ciptaland Blok Mawar"
   - Klik **"Insert"**

### Metode 2: Import Langsung di Elementor Editor

1. **Buka Elementor Editor**
   - Edit halaman dengan Elementor

2. **Buka Template Library**
   - Klik icon **Folder** di panel widget

3. **Import Template**
   - Klik icon **Upload** (panah ke atas)
   - Pilih file: `elementor-hut-ri-81-template.json`
   - Template akan otomatis ter-import dan inserted

## 🔧 Troubleshooting

### Template Tidak Muncul Setelah Import
- Refresh halaman WordPress
- Clear browser cache
- Pastikan Elementor versi terbaru

### Layout Tidak Full Width
- Check Page Settings → Layout → Elementor Full Width
- Check container settings → Content Width → Full Width

### Container Tidak Berfungsi
- Pastikan Elementor Pro aktif (untuk fitur Container)
- Update Elementor ke versi terbaru
- Enable Flexbox Container di Elementor Settings → Features
